Q1. Difference between local storage and session storage?

Local storage and session storage is similar thing for insert and get data, the
main difference is local storage data doesn't expire. Where session storage data
expire after close the page or browser.

I). Local Storage Capacity is 5MB/10MB

I). The storage capacity of seesion storage is 5MB.

Block Scope

From ES6 Javascript introduced a block scope, let and const, these variable decare in a function or block Under {}

Q3.Event Loop In Javascript?

We Know JavaScript is a synchronous, blocking, single-threaded language.This means only one operation can be in progress
at a time.But event loop in a mechanism that allows Javascript to perform no-blocking operations.When an asynchronous
task is started the event loop will start running.When the task is completed the event loop will again check for any other tasks
that need to be performed.This process will continue until all tasks have been completed.
