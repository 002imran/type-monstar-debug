Q1. Difference between local storage and session storage?

Local storage and session storage is similar thing for insert and get data, the
main difference is local storage data doesn't expire. Where session storage data
expire after close the page or browser.

Loacl Storage:

I). Local Storage Capacity is 5MB/10MB
II). It is not session based it must be deleted via javascript or manually.
III).The client can only read local storage.

Session Storage:

I). The storage capacity of seesion storage is 5MB.
II). Its session based and works per window or tab, data will vanised after close the page or tab.
III). The client only can read the session storage
